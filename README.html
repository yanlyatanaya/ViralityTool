🇬🇧 User Guide (English)

1. Prerequisites

Install Python 3.10+ and FFmpeg on your system. FFmpeg is mandatory for video cropping.

2. Installation

pip install streamlit moviepy faster-whisper openai python-dotenv

3. Configuration

Open processor.py and enter your DeepSeek API Key in the dedicated variable.

4. Launch

python -m streamlit run main.py --server.maxUploadSize 5000
